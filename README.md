<h2>📊 DataCo Smart Supply Chain Analysis Project</h2>

<p>
This project focuses on analyzing the <strong>DataCo Smart Supply Chain dataset</strong> 
to extract meaningful insights and support data-driven decision-making.
</p>

<h3>🚀 Project Overview</h3>
<p>
The goal of this project is to apply end-to-end data analytics techniques, 
starting from raw data processing to building interactive dashboards.
</p>

<h3>🧩 Key Steps & Technologies</h3>

<ul>
  <li>
    <strong>Data Cleaning & Transformation</strong>
    <ul>
      <li>Performed using Power Query in Power BI</li>
      <li>Handling missing values, data types, and restructuring datasets</li>
    </ul>
  </li>

  <li>
    <strong>Data Modeling</strong>
    <ul>
      <li>Designing a Star Schema</li>
      <li>Creating fact and dimension tables to optimize performance and enable efficient analysis</li>
    </ul>
  </li>

  <li>
    <strong>Exploratory Data Analysis (EDA)</strong>
    <ul>
      <li>Conducted using Python</li>
      <li>Libraries used: Pandas, NumPy, Matplotlib, Seaborn</li>
      <li>Identifying trends, patterns, and anomalies</li>
    </ul>
  </li>

  <li>
    <strong>Database & Querying</strong>
    <ul>
      <li>Using SQL for data extraction and manipulation</li>
      <li>Writing queries to support analysis and validation</li>
    </ul>
  </li>

  <li>
    <strong>Data Visualization</strong>
    <ul>
      <li>Building interactive dashboards using Power BI</li>
      <li>Providing insights into:
        <ul>
          <li>Sales performance</li>
          <li>Supply chain efficiency</li>
          <li>Customer behavior</li>
          <li>Delivery performance</li>
        </ul>
      </li>
    </ul>
  </li>
</ul>

<h3>📈 Project Outcome</h3>
<p>
The final output is a comprehensive Power BI Dashboard that helps stakeholders:
</p>

<ul>
  <li>Monitor KPIs</li>
  <li>Identify bottlenecks</li>
  <li>Improve operational efficiency</li>
</ul>
